## ⚙️ Getting Started

### Prerequisites
- A modern web browser (**Google Chrome recommended** for full Web Speech API support)
- No installations or dependencies required

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/jarvis-advanced.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd jarvis-advanced
   ```

3. **Open the main file in your browser**
   ```bash
   # Simply double-click the file, or open via terminal:
   open jarvis_advanced.html        # macOS
   start jarvis_advanced.html       # Windows
   xdg-open jarvis_advanced.html    # Linux
   ```

> ⚠️ **Note:** Microphone access requires running from a local or HTTPS server. If voice input doesn't work, try serving via a local server:
> ```bash
> npx serve .
> ```

## 🎮 Usage

- **Type** your command in the input box and press Enter or click Send
- **Speak** your command using the 🎤 microphone button
- Use the **Quick Command buttons** for instant actions:
  - 🕒 **Time** — Get the current time
  - 🌐 **Google** — Open Google
  - 😂 **Joke** — Hear a joke
  - ▶️ **YouTube** — Go to YouTube
- Adjust **voice rate and pitch** using the sliders in the control panel
